
# Course-Registration

This  registration website project is designed specifically for managing programming-related courses and workshops.


## Features

- Select Button For Adding Info In Cart
- Cart Information Managment
- Toasting on different condition


## "Select Button" for adding info in Cart By Using "useState"

![Alt](https://i.ibb.co/1RZZKLg/Screenshot-2023-09-15-130422.png)
"Selected Button" likely a particular button element in this webpage that is influenced or controlled by a JavaScript event handler function.

Additionally, the selected item's information is being managed using the `useState` hook, and this information is displayed in the cart section of this webpage. 


## Cart Information Managment By Using "useState"

![Alt](https://i.ibb.co/KFGhmHH/Screenshot-2023-09-15-135200.png)

The "Cart Section" refers to a specific section of the user interface that displays information about selected iinformation or credit hour. This information is dynamically updated based on user interactions, such as adding credit hour's, decreasing credit hour's or adding titles from card to cart. The cart section information is controlled by a JavaScript event-handler function, and the information displayed is managed and updated using the `useState` hook in React.
## Toast/SweetAlert By Using Condition Of "useState" Value

![Alt](https://i.ibb.co/mCMFmRw/Screenshot-2023-09-15-142245.png)

Toast/Alert is a user interface element used to display non-intrusive messages or notifications to the user. These messages typically provide feedback about a particular action or event and disappear after a short period. In this context, the toast is controlled by a JavaScript event-handler function and displayed based on a `useState` hook value.
